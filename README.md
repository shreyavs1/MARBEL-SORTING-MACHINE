# Marble Sorting Machine

## Overview

The **Marble Sorting Machine** is an automated system designed to identify and sort marbles based on predefined characteristics such as color, size, or material. The project demonstrates the integration of embedded systems, sensors, actuators, and automation principles to improve sorting accuracy while reducing manual effort.

This project was developed as part of an engineering coursework to explore real-world industrial automation concepts.

---

## Features

- Automated marble detection
- Real-time sorting mechanism
- High sorting accuracy
- Continuous operation
- Compact and efficient design
- Low-cost automation solution

---

## Problem Statement

Manual sorting of objects is time-consuming, inconsistent, and labor-intensive. This project aims to automate the sorting process using sensors and embedded control, improving efficiency and reducing human intervention.

---

## Objectives

- Design an automated marble sorting system
- Improve sorting speed and accuracy
- Reduce manual effort
- Demonstrate embedded system applications in automation

---

## Working Principle

1. A marble enters the system through the input channel.
2. Sensors detect the marble's properties.
3. The controller processes the sensor data.
4. Based on the detected characteristics, the controller activates the sorting mechanism.
5. The marble is directed into its corresponding collection bin.
